## Calculator Using Java AWT##

A simple Calculator application developed using Java AWT (Abstract Window Toolkit). This project performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division through a graphical user interface (GUI).

## Features

 Addition
 Subtraction
 Multiplication
 Division
 Clear Button
 GUI-based Calculator
User-friendly Interface

## Technologies Used

 Java
 AWT (Abstract Window Toolkit)
 Event Handling
 OOP Concepts

## Concepts Used
Frame
Panel
GridLayout
Button
TextField
ActionListener
Event Handling
Switch Case

## Output
The calculator takes two numbers as input and performs arithmetic operations when the operator and equal button are pressed.
Example:
12 + 5 = 17
